# music_player
# 简易的音乐播放器

# 开发环境
1.npm install

2.npm run dev

3.npm run start

# 打包
1.npm run make

## Usage 使用方法
1.导入包含mp3 的文件夹
2.点击播放

# 快捷键
1.ALT + -> 下一首

2.ALT + <- 上一首

3.ALT + BlankSpace((空隔) ： 暂停(pause) 播放(replay)
